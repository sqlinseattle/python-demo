# python-demo

Small demo workspace that includes a script to query the current Bitcoin price and print it to the console.

## Requirements
- Python 3.8+ (or your system Python)
- See `requirements.txt` for dependencies (requests)

## Run
Install dependencies and run the script:

```powershell
python -m pip install -r requirements.txt
python print_btc.py
```

The script prints the current Bitcoin price (USD) or an error message if the fetch fails.
